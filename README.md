# TZG API 主页
TZG API是本人由于项目的原因需要在公网使用的一些接口，不涉及到保密问题的接口都会在这里进行开放。这个接口是公开的，任何人都可以进行调用。   
如果你对这个接口有任何意见，请前往[留言板]("https://zihangu.com/?page_id=175")进行反馈。
## 协议
* 本站API所有内容均不开源，属于非开源项目。
* 禁止利用反向代理对本站接口进行反代。
* 我们将尽量保持本站服务的可用性，但出现服务异常的情况我们概不负责。
