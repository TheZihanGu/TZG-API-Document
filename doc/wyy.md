## 数据介绍
从我们的数据库中随机输出一句网抑云语录。
## 获取方式
HTTP GET
```
https://api.tzg6.com/api/wyy
```

## 返回参数
``` id ```: 语句编号   
``` text ```: 随机选取到的祖安语录   
## 返回数据示例
```
[
    {
        "id":13,
        "text":"爱要两情相悦，否则，所有的喜欢都是心酸。"
    }
]
```

## 备注
若接口出现异常，将只返回``` status ```参数。   
``` status ```参数仅在接口出现异常时返回。
